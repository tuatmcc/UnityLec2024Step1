# Unity 講習会 2024 入門編

## 前提

Unity をインストール済み
任意の IDE がある(Visual studio, Rider など)

## 題材

玉転がしゲームを作る

## 学ぶこと

* Transform
  * 座標移動
    * Quaternion は詳しく触れない
      * `Quaternion.Euler` で済ませる
  * 回転
* Rigidbody
  * 自由落下
  * AddForce
* Collider
  * SphereCollider 3D
    * OnCollision
    * OnTrigger
* Prefab
* Audio
  * AudioSource
  * AudioClip
  * AudioListener
* Camera
* Script
  * MonoBehaviour
  * Update
  * Start
